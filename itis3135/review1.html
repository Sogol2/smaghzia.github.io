Checklist

  3. Design
    1. ✔ Page has sufficient contrast/font sizing so as to be easy to read
    2. ✔ Page has site colors and fonts using the standard css file, unless this assignment requires a unique/standalone/embedded one
    3. CRAP
      1. ✔ Contrast
      2. ✔ Repetition
      3. ✔ Alignment
      4. Proximity
        • ✘ Some related items (like service times and mission text) could be grouped a little closer to emphasize their connection.
  4. Page has within it:
    1. ✔ Header
    2. ✔ Main
    3. ✔ Footer
    4. ✔ (optional) nav for main menu between header and main
  5. Header has site/brand header with text in an h1
    1. ✔ If this is a homework page it's likely something like Student Name * Course ID Course Name
    2. ✔ If this is a "branded page" it should be the brand/company
    3. ✔ DO NOT INCLUDE THE PAGE NAME IN THE HEADER
    4. ✔ Understand the difference between head, header, and heading
  6. ✔ Main starts with name of page as an h2 and does not include name of site/brand
  7. Page has brand tagline somewhere (make up a slogan and include it on every page in a consistent location)
    • ✘ I don’t see a short tagline separate from the mission statement; adding a slogan and repeating it in header or footer would strengthen the brand.
  8. Page has footer with
    1. ✔ Menu for user's pages
    2. ✔ Page has html validation button/link and validates
    3. ✔ Page has css validation button/link and validates
  9. ✔ Page has specific requirements for this assignment (make a checklist from that assignment and make sure you've covered everything there). If there is a conflict between what you see here and in the instructions, ask on the class forums about it.
